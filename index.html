QCLean: Remove Facebook Ads, Suggested Pages and Posts

This is an open source web browser extension that to hide ads and sponsored posts on news feed of Facebook, make you have a clean Facebook interface and focus on what you really need.

Source code: https://github.com/qcl/QCLean

If you want to report issue, you can report here: https://github.com/qcl/QCLean/issues

* v0.5.9.7: Fix issue of hiding sponsored post on newsfeed.

* v0.5.9.6: Fix bugs on Facebook new layout.

* v0.5.9.5: Fix bug and hide new type of sponsored posts for new Facebook layout.
